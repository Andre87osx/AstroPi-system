📝 LOCATION DEI FILE
═══════════════════════════════════════════════════════════════════════════════

I file INDI helper sono disponibili in:
   ~/.local/share/astropi/bin/

Per accedervi da ovunque nel sistema:
   $ bash ~/.local/share/astropi/bin/quick-fix-indi.sh
   $ bash ~/.local/share/astropi/bin/check-indi-deps.sh
   $ bash ~/.local/share/astropi/bin/fix-indi-dependencies.sh
   $ bash ~/.local/share/astropi/bin/verify-indi-fix.sh

Per accedervi più facilmente, puoi aggiungere un alias:
   alias quick-fix-indi='sudo bash ~/.local/share/astropi/bin/quick-fix-indi.sh'
   alias check-indi-deps='bash ~/.local/share/astropi/bin/check-indi-deps.sh'
