Il problema: ❌
   Compilazione di INDI fallisce su Debian Buster per dipendenze mancanti
   nei repository archiviati (Buster EOL: 30 Giugno 2024)

La soluzione: ✅
   Sistema intelligente di fallback + repository Raspberry Pi + 3 script helper

📦 FILE CREATI (4 nuovi)
═══════════════════════════════════════════════════════════════════════════════

1. bin/fix-indi-dependencies.sh
   • Pre-risoluzione completa delle dipendenze
   • Installazione intelligente di pacchetti critici e opzionali
   • Risoluzione automatica di dipendenze rotte
   • Log dettagliato
   📌 Uso: sudo bash bin/fix-indi-dependencies.sh

2. bin/check-indi-deps.sh
   • Verifica rapida dello stato delle dipendenze
   • Output a colori (verde=ok, rosso=critico, giallo=opzionale)
   • Nessuna modifica al sistema
   📌 Uso: bash bin/check-indi-deps.sh

3. bin/quick-fix-indi.sh
   • Fix rapido in una sola esecuzione
   • Combinazione di step essenziali
   • Output formattato e comprensibile
   📌 Uso: sudo bash bin/quick-fix-indi.sh

4. bin/99-indi-buster-archive.conf
   • Configurazione APT ottimizzata
   • Disabilita verifiche di validità per archivi
   • Consente repository non firmati
   📌 Installazione: sudo cp bin/99-indi-buster-archive.conf /etc/apt/apt.conf.d/

🔄 COME USARE
═══════════════════════════════════════════════════════════════════════════════

FLUSSO CONSIGLIATO:
───────────────────

Step 1: Prepara Sistema (una volta sola)
   $ ./bin/AstroPi.sh
   Seleziona: "System Pre Update"
   ⏱️  Tempo: 5-15 minuti

Step 2: Pre-risolvi Dipendenze (consigliato)
   $ sudo bash bin/quick-fix-indi.sh
   ⏱️  Tempo: 5-10 minuti

Step 3: Verifica (opzionale)
   $ bash bin/check-indi-deps.sh
   ⏱️  Tempo: <1 minuto

Step 4: Compila INDI
   $ ./bin/AstroPi.sh
   Seleziona: "Check INDI"
   ⏱️  Tempo: 30-60 minuti (compilazione lunga)

ALTERNATIVA RAPIDA:
───────────────────

Se System Pre Update già eseguito:
   $ sudo bash bin/quick-fix-indi.sh
   $ ./bin/AstroPi.sh → Check INDI

SOLO VERIFICA:
──────────────

Controlla stato senza modifiche:
   $ bash bin/check-indi-deps.sh
